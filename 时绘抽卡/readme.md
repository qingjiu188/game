# 时绘抽卡系统

一个基于《时空中的绘旅人》的网页版抽卡模拟器。

## 📁 文件说明

- `index.html` - 主页面，显示所有卡池列表
- `permanent-pool.html` - 常驻卡池页面
- `limited-pool.html` - 限定卡池页面（通用模板）
- `game-data.js` - 游戏数据（所有卡牌信息）
- `gacha-system.js` - 抽卡系统核心逻辑

## 🎮 功能特性

### 1. 卡池系统
- **常驻池**：包含所有常驻SSR、SR、R卡
- **限定池**：32个限定卡池，每个包含当期限定SSR + 常驻池所有卡牌

### 2. 抽卡机制

#### 常驻池规则
- SSR概率：2%
- SR概率：10%
- R概率：88%
- 80抽保底SSR
- 十连抽必出至少一张SR

#### 限定池规则
- 基础概率与常驻池相同
- 80抽保底SSR（可能是常驻或限定）
- 160抽必出限定SSR
- 十连抽必出至少一张SR
- 若多个限定SSR，概率均等

### 3. 特色功能
- ✨ **翻牌特效**：SSR卡牌具有金色光晕动画效果
- 📊 **保底进度显示**：实时显示距离保底还剩多少抽
- 📜 **抽卡记录**：完整的抽卡历史，支持分页查看（每页20条）
- 🔄 **重置档案**：清空抽卡记录和保底计数
- 💾 **本地存储**：使用localStorage保存数据，刷新页面不丢失
- 🎯 **快速切换**：右上角导航栏可快速切换不同卡池

### 4. 视觉设计
- 清淡柔和的配色方案
- SSR：金色渐变 + 发光动画
- SR：淡黄色渐变
- R：淡紫色渐变
- 响应式设计，支持移动端

## 🚀 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 点击任意卡池进入抽卡页面
3. 点击"单次抽卡"或"十连抽卡"按钮开始抽卡
4. 查看翻牌动画和抽卡结果
5. 在下方查看抽卡记录
6. 可随时点击"重置档案"清空数据

## 📋 技术实现

### 核心技术
- HTML5
- CSS3（渐变、动画、3D变换）
- JavaScript（ES6+）

### 关键算法
- **随机数生成**：Math.random()
- **保底机制**：计数器累加，达到阈值强制触发
- **十连SR保底**：前9抽统计，第10抽判断是否需要强制SR
- **数据持久化**：localStorage API

### 数据结构
```javascript
{
    poolType: 'permanent' | 'limited',
    poolName: string,
    pityCount: number,        // SSR保底计数
    limitedPityCount: number, // 限定SSR保底计数
    history: array            // 抽卡历史
}
```

## 🎨 设计亮点

1. **3D翻牌效果**：使用CSS transform-style: preserve-3d 实现真实的翻牌动画
2. **SSR光晕动画**：@keyframes实现呼吸灯效果的金色光晕
3. **渐变色系**：每个稀有度都有独特的渐变背景
4. **卡片式布局**：现代化的圆角卡片设计
5. **流畅过渡**：所有交互都有平滑的过渡动画

## 📝 注意事项

1. 数据保存在浏览器本地，清除浏览器缓存会丢失数据
2. 每个卡池的数据独立存储
3. 不同卡池之间的保底次数不继承
4. 建议在Chrome、Firefox、Safari等现代浏览器中使用
